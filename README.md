# AIMemory Platform

AIMemory is an AI-powered experiment intelligence platform designed to track, analyze, and reason about machine learning experiments.

## Tech Stack
- PostgreSQL (OLTP database)
- FastAPI (backend API)
- Streamlit / Web UI (frontend dashboard)
- Vector embeddings + AI insights (planned)
- Cloud deployment (planned)

## Goals
- Track ML experiments and datasets
- Analyze training metrics
- Provide AI-driven experiment insights
- Serve as a personal data platform project
